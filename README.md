# Number Guessing Game

A fun and interactive **Number Guessing Game** where the player tries to guess a randomly generated number within a given range. The game provides instant feedback to guide the player until the correct number is guessed. This project is ideal for beginners learning JavaScript logic, conditions, and user interaction.

The goal is to guess the number using logic and feedback in as few attempts as possible.
